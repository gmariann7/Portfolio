MTC vs. MTV

A Model View Controller (MVC) is a pattern or framework that relates the user interface to the data models. The pattern is split into three categories which are the model, view, and controller. The model stores data, the view uses the user input to display the data, and the controller controls the flow between the view and the model. A framework is a collection of libraries while libraries are a collection of certain functions. Using a framework would be better then creating an app yourself because it is much more faster and efficient because having a framework allows you to not have to create everything.
